1.DOM 操作演示

Document Object Model

插件 emmet 学习

DOM 定义了表示和修改文档所需的对象、这些对象的行为和属性以及这些对象之间的关系。DOM 对象即为宿主对象，由浏览器厂商定义，用来操作html 和 xml功能的一类对象的集合(不能修改 CSS 样式表，可以改变行间样式，即通过改变 html 间接修改 css)。也有人称 DOM 是对 HTML 以及 XML 的标准编程接口。
